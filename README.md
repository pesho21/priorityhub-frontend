## Стартиране на Frontend-а

Ако искате да стартирате само и единствено frontend-а, използвайте тези команди:

```
# Clone repo
git clone https://github.com/pesho21/priorityhub-frontend.git

# Project setup
npm install

# Run the frontend
npm start

```
